# C++ 색 선호도 조사 차트

배열과 클래스를 이용하여
학생들의 색 선호도를 백분율로 출력하는 프로그램을 만들었다.

---

# 코드 설명

## 사용한 개념

- 클래스(Class)
- 생성자(Constructor)
- 소멸자(Destructor)
- 배열(Array)
- 반복문
- 백분율 계산
- iomanip 출력 포맷

---

# 전체 코드

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

class Chart {
private:
    string title;
    int data[3];
    string colors[3];

public:
    Chart(string t) {
        title = t;
        colors[0] = "빨강";
        colors[1] = "파랑";
        colors[2] = "노랑";

        for (int i = 0; i < 3; i++)
            data[i] = 0;
    }

    int getNumberOfColors() {
        return 3;
    }

    string getColor(int index) {
        return colors[index];
    }

    void add(int index, int count) {
        data[index] = count;
    }

    int getData(int index) {
        return data[index];
    }

    void draw() {
        cout << title << "********************" << endl;

        int total = data[0] + data[1] + data[2];

        cout << fixed << setprecision(4);

        for (int i = 0; i < 3; i++) {
            double percent = 0.0;

            if (total > 0)
                percent = (double)data[i] / total * 100;

            cout << colors[i] << ": "
                 << percent << "%" << endl;
        }

        cout << "**************************************" << endl;
    }

    ~Chart() {
        cout << "차트 데이터는 저장되지 않고 사라집니다." << endl;
    }
};

int main() {
    Chart chart("색 선호도 조사 차트");

    for (int i = 0; i < chart.getNumberOfColors(); i++) {

        cout << chart.getColor(i)
             << "이 좋은 학생>>";

        int count;

        cin >> count;

        chart.add(i, count);
    }

    chart.draw();
}
```

출력:

```text

**************************************
```

# 느낀 점

단순 배열이 아니라
클래스로 데이터를 관리하니
구조를 정리하기 쉬웠다.

또한 `setprecision()`을 이용하여
실수 출력 형식을 조절하는 방법도 익힐 수 있었다.
